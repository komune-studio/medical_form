# Read Indonesia Dashboard
![version](https://img.shields.io/badge/version-1.2.2-blue.svg) ![license](https://img.shields.io/badge/license-MIT-blue.svg)

This website is an admin dashboard for Read-Indonesia using Argon Dashboard React. 

## Argon Design System
Argon is An user-friendly, open source and beautiful design system based on Bootstrap 4 using React, Reactstrap and create-react-app.

## Quick start

- `npm install`
- `npm start`

## Documentation

- Argon: https://demos.creative-tim.com/argon-dashboard-react/#/documentation/overview
- Ant-Design: https://antdv.com/components/overview/
- Iconify-Icon (Search for Icon): https://icon-sets.iconify.design/
- React-Quill: https://www.npmjs.com/package/react-quill
